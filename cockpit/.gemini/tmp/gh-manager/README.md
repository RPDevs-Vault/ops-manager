# T430 GitHub Actions Runner

This repository contains the configuration to deploy self-hosted GitHub Actions runners on the T430 hardware node.

## Configuration
The runners are deployed using Docker Compose and utilize the standardized builder image generated by the `container-manager` repository.

### Runners
1.  **Main Builder (`t430-linux-builder`):**
    - **Resource Limits:** 3 CPUs, 6GB Memory
    - **Data Path:** `/mnt/data/github-runners/t430-linux-builder/work`
    - **Labels:** `self-hosted,linux,t430`
2.  **Lite Builder (`t430-lite-linux-builder`):**
    - **Resource Limits:** 1 CPU, 2GB Memory
    - **Data Path:** `/mnt/data/github-runners/t430-lite-runner/work`
    - **Labels:** `self-hosted,linux,t430,lite`

## Deployment
Clone this repository to the T430 host and execute:

```bash
docker compose up -d
```

*(Ensure the `.env` file containing `GH_PAT` is present in the deployment directory).*
